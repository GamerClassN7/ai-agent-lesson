# T-005: UX/UI Optimalizace kalendáře

## Cíl
Převést stávající UI na standardní kalendářové rozhraní, které je intuitivní a použitelné pro všechny uživatele zvyklé na běžné kalendářové systémy (Google Calendar, Outlook atd.).

## Navržené změny

### 1. Navigace a ovládání
- [ ] Přidat tlačítko „Dnes“ (Today) pro rychlý návrat k aktuálnímu datu.
- [ ] Zjednodušit přepínání pohledů (Týden/Měsíc) pomocí standardnějších UI prvků.
- [ ] Vylepšit vizuální hierarchii v headeru kalendáře.

### 2. Kalendářový grid
- [ ] Zvýraznit víkendy (např. jemně odlišné pozadí).
- [ ] Přidat čísla týdnů (volitelně/při týdenním pohledu).
- [ ] Vylepšit zobrazení eventů přímo v buňkách (v měsíčním pohledu aspoň náznaky/text, ne jen tečky).
- [ ] Aktivní den by měl být jasně odlišen od dneska.

### 3. Event Panel (Sidepanel)
- [ ] Lepší struktura výpisu eventů pro vybraný den.
- [ ] Možnost přidat event (placeholder/UI prvek).
- [ ] Vylepšit vizuální oddělení kategorií.

### 4. Responzivita a Přístupnost
- [ ] Zajistit, aby kalendář vypadal dobře i na menších displejích (stohování panelu).
- [ ] Vylepšit barvy pro lepší kontrast v dark/light režimech.
- [ ] Keyboard navigace (základní podpora).

## Akceptační kriteria
- Uživatel se umí rychle vrátit na dnešní den.
- V měsíčním pohledu je vidět víc informací o eventech než jen tečky.
- Celkový vzhled působí moderně a profesionálně.
